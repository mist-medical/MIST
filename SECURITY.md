# Security Policy

## Supported versions

MIST is research software under active development. Security fixes are applied
to the latest release of `mist-medical` on PyPI and to the `main` branch.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report them privately through GitHub's
[private vulnerability reporting](https://docs.github.com/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability)
(the repository's **Security** tab → **Report a vulnerability**), or email the
maintainers at aecelaya@rice.edu.

Please include enough detail to reproduce the issue. We will acknowledge your
report as soon as possible and keep you informed of remediation progress.
